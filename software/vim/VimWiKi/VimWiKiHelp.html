注意事项

  • 转换的网页调用的style.css为绝对路径，如果可以调用指定路径会更灵活
  • 调用绝对路径的文件转换html后，为绝对路径，导致复制到其他地方图片失效
  • 由于VimWiki编译成html编译，默认使用文件名作为标题，但文件名为中文时，
    html输出标题为乱码，导致脚本打开文件出现解码错误报警
    1. 目前中文文件名时，强制使用 %title title—name 指定一个非中文标题名
    2. 避免使用中文标题名

常用快捷键

****** ********* ***************************************************************
模式 快捷键 说明
NORMAL \ww 开启wiki并到首页
NORMAL \wr Rename wiki page you are in
NORMAL \tt Toggle checkbox of a list item on/off
NORMAL F4 编译当前页
NORMAL <S-F4> 编译所以页---慎用 会导致网页乱码 使用F4单独编译
NORMAL \wh Convert current wiki page to HTML
NORMAL \whh Convert current wiki page to HTML and open it in the webbrowser
NORMAL <CR> Follow/create wiki link (create target wiki page if needed)
NORMAL = Add header level. Create if needed
NORMAL - Remove header level
NORMAL Backspace 返回父页
NORMAL gl* symbol of the current item to *
NORMAL gl# symbol of the current item to #
NORMAL gl- symbol of the current item to -
NORMAL gla symbol of the current item to a)
NORMAL gli symbol of the current item to i)
NORMAL glI symbol of the current item to I)
NORMAL Tab 跳到下一个维基词条或链接
NORMAL Shift-Tab 跳到上一个维基词条或链接
NORMAL [[ | Go to the previous header in the buffer 上一个标题
NORMAL ]] Go to the next header in the buffer 下一个标题
     

DiaryNote快捷键

****** ********* ***************************************************************
模式 快捷键 说明
NORMAL <C-Up> Open the previous day's diary link if available
NORMAL <C-Down> Open the next day's diary link if available
NORMAL \wi opens diary index file of the first wiki from
NORMAL \w\w opens diary wiki-file for today in the first wiki
NORMAL \w\t opens diary wiki-file for today
NORMAL \w\y opens diary wiki-file for yesterday
NORMAL \w\m opens diary wiki-file for tomorrow

Tables快捷键

****** ********* ***************************************************************
模式 快捷键 说明
NORMAL gqq Format table
NORMAL <A-Left> Move current table column to the left
NORMAL <A-Right> Move current table column to the right
NORMAL vac 可视模式选择一列表格
INSERT <CR> Go to the table cell beneath the current one, create
a new row if on the last one
INSERT <Tab> Go to the next table cell, create a new row if on the
last cell.
INSERT <S-Tab> Go to the next table cell, create a new row if on the
last cell.
