打开文件末尾带有^M的解决办法

而在linux下打开windows编辑过的文件，就会在行末尾显示^M;

  1. 在windows下的文本文件的每一行结尾，都有一个回车('\n')和换行('\r')
  2. 在linux下的文本文件的每一行结尾，只有一个回车('\n');
  3. 在Mac下的文本文件的每一行结尾，只有一个换行('\r');
  • %s/^M$//g
    解释：% 指匹配整个文件，s 是置换的意思，^M 注意要用 Ctrl + V Ctrl + M 来输入，
    M 后面的 $ 代表匹配行尾的内容，最后的 g 则表示每行中匹配到的内容都要置换；

编程字体安装配置

在gvim中配置中文字体:set gfw=幼圆:h10.5:cGB2312目前无效
  1. 字体安装
    • 下载ttf字体格式文件
    • 打开ttf文件后选择安装
  2. 字体配置
    • :set guifont 查看当前使用自己
    • :set guifont=* 配置字体
    • _vimrc设置字体格式 set guifont=字体:大小:cANSI
  3. 字体概述
    1. Hack字体
      • set guifont=Hack:h10:cANSI
      • Hack是一种专门用来显示编程语言的字体，基于Bitstream Vera和DejaVu的开源字体
        继承Mono中0oO，iLI1清晰可见的特点，并进一步优化0字体样式
    2. Mono 字体
      • set guifont=Bitstream_Vera_Sans_Mono:h10:cANSI
      • Mono是一种专门用来显示编程语言的字体，其0oO，iLI1清晰可见的特点
