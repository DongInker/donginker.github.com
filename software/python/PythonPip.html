快速收集安装Python依赖插件

  • 先安装wheel处理插件
    • pip install wheel
  • 导出项目依赖插件
    1. 在项目根目录运行命令行----(在项目根目录 按Shift键 右击选择命令行)
    2. 行 pip freeze > requirements.txt
  • 安装项目依赖插件
    1. 在项目根目录运行命令行
    2. 行 pip install -r requirements.txt

Python安装插件----pip

  1. pip升级
    • python -m pip install --upgrade pip
  2. pip命令格式
    • pip <command> [options]
  3. pip帮助命令
    • pip help
  4. 更新pip安装包程序命令
    • python -m pip install -U pip
  5. 安装一个安装包（可以指定版本安装）
    • pip install SomePackage
  6. 删除一个安装包
    • pip uninstall SomePackage
  7. 更新一个安装包命令
    • pip install --upgrade SomePackage
  8. 查看已安装的数据包列表
    • pip list
  9. 检查数据包当前安装包版本及最新版本
    • pip list --outdated
  10. 查看安装包所以文件
    • pip show --files SomePackage
  11. 安装下载的whl包
    • pip install C:\Users\xxx\Downloads\python_dateutil-2.5.3-py2.py3-none-any.whl
